# FitGuide — AI-Powered Fitness Recommendation System

A full-stack AI fitness coaching web application that generates personalized 
workout plans through conversational AI. Built with Next.js 15, TypeScript, 
MongoDB, and Groq AI API.

## Live Demo
https://fitguide-omega.vercel.app/

## Tech Stack
- **Frontend:** Next.js 15, React 19, TypeScript, Tailwind CSS
- **Backend:** Next.js API Routes, Node.js
- **Database:** MongoDB, Mongoose
- **AI:** Groq AI API — conversational workout plan generation
- **Authentication:** Clerk (user accounts, session management)
- **Deployment:** Vercel

## Features
- Conversational AI coach that asks about your preferences and creates custom workout plans
- Multiple workout types — HIIT, strength training, yoga, core, and more
- Live workout tracking — mark completed exercises in real time
- Workout calendar — visualize your fitness history and consist
